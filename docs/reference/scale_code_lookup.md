# Correspondence table for CTPP codes, table type, and scale

Correspondence table for CTPP codes, table type, and scale

## Usage

``` r
scale_code_lookup(scale, table_code)
```

## Arguments

- scale:

  "county", "place", or "tract" for residence/workplace tables;
  "county-county", "place-place", "place-county", "county-place", or
  "tract-tract" for flow tables

- table_code:

  requested data table as string, e.g. "A302103"

## Value

correspondence table
