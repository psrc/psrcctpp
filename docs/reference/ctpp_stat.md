# CTPP summary statistics

CTPP summary statistics

Generate CTPP sums

## Usage

``` r
psrc_ctpp_sum(df, group_vars = NULL, incl_na = TRUE)
```

## Arguments

- df:

  result of get_psrc_ctpp function

- group_vars:

  grouping variables, original or added

- incl_na:

  whether to include NA lines

## Value

A table with the variable names and labels, summary statistic and margin
of error
