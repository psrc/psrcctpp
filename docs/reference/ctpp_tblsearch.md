# Search CTPP table codes

Search CTPP table codes

## Usage

``` r
ctpp_tblsearch(prefix, regex, year)
```

## Arguments

- prefix:

  2-character pattern: see
  [`vignette("psrcctpp_basics", package = "psrcctpp")`](https://psrc.github.io/psrcctpp/articles/psrcctpp_basics.md)

- regex:

  string pattern to match table description

- year:

  last of 5-year CTPP span, e.g. 2016 for ctpp1216 survey

## Value

data table
